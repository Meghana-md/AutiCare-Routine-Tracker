## 🧠 Module: Routine Coach + Smart Reminders

The **Routine Coach** helps caregivers manage daily routines for children by scheduling and tracking important activities such as **therapy sessions, meals, and medication**.  
It provides reminders and progress tracking to ensure tasks are completed on time.

### 🎯 Purpose
Children benefit from consistent routines. This module helps caregivers:
- Plan daily activities
- Receive reminders for important tasks
- Track completed and missed routines

### 🖥️ Key Features

**1. Add Routine**
- Create tasks such as Therapy, Meal, or Medication  
- Set task time  
- Option to repeat daily

**2. Today’s Routine**
- Displays all tasks scheduled for the day
- Tasks sorted by time
- Action buttons:
  - ✅ Done
  - ⏭ Skip
- Shows daily progress and task status

**3. Smart Reminder**
At the scheduled time, a reminder popup appears.

Options:
- ✅ Done
- ⏰ Remind in 10 minutes

### 🧠 Smart Logic
- If **Remind in 10 minutes** is selected → reminder is rescheduled.
- If there is **no response within 30 minutes** → task is marked as **Missed**.

### 📊 Tracking
The system tracks:
- Completed tasks
- Missed tasks
- Daily routine progress

### 📌 Summary
This module acts as a **digital routine assistant**, helping caregivers maintain structured daily schedules and improve routine adherence.
